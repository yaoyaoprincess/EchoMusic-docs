### 参与贡献

欢迎参与 EchoMusic 的开发！贡献方式：

- **提交 Issue**：报告 Bug 或提出功能建议
- **提交 Pull Request**：修复 Bug 或实现新功能
- **改进文档**：补充和完善项目文档

在提交 Issue 时，请：
1. 使用清晰的标题描述问题
2. 提供复现步骤和环境信息（操作系统、EchoMusic 版本）
3. 附上相关的日志输出或截图
4. 检查是否已有相同问题的 Issue
