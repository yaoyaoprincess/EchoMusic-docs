### Contributing

We welcome contributions to EchoMusic! Ways to contribute:

- **Submit Issues**: Report bugs or suggest features
- **Submit Pull Requests**: Fix bugs or implement features
- **Improve Documentation**: Supplement and improve project docs

When submitting an issue, please:
1. Use a clear title to describe the problem
2. Provide reproduction steps and environment info (OS, EchoMusic version)
3. Attach relevant log output or screenshots
4. Check if a similar issue already exists
