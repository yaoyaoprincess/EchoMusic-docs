# 功能特性

EchoMusic 是一款专为桌面端打造的第三方酷狗概念版音乐播放器，集美学设计、强大功能与极致体验于一体。

<div class="feature-cards">

<div class="feature-card">

### 🎨 极致美学

精心适配桌面端布局，支持深浅色模式，兼顾信息密度与沉浸式体验。每一处细节都经过打磨，让音乐播放成为视觉享受。

</div>

<div class="feature-card">

### 🔒 数据安全

官方服务器直连，所有数据不经过第三方服务器。你的登录信息、听歌记录、收藏列表都掌握在自己手中。

</div>

<div class="feature-card">

### 🎵 进阶播放能力

播放队列管理、列表循环/单曲循环/随机播放模式切换、音量调节、进度拖动、倍速播放（0.5x - 2.0x）、淡入淡出切歌——专业级播放体验。

[了解更多 →](/guide/playback)

</div>

<div class="feature-card">

### 🔍 多维探索与搜索

支持歌曲、歌手、专辑、歌单全方位搜索。首页推荐、排行榜、新歌速递、热门歌手，让发现音乐变成一种享受。

[了解更多 →](/guide/music-discovery)

</div>

<div class="feature-card">

### 📝 歌词系统

支持 LRC/YRC 逐字歌词解析，精准滚动同步。全屏歌词模式与桌面歌词悬浮窗，让你沉浸在每一段旋律中。

[了解更多 →](/guide/lyrics)

</div>

<div class="feature-card">

### 🎚️ 音频增强

10 段 EQ 均衡器自由调节，基于 LUFS 的音量均衡，蝰蛇母带/全景声/超清等多种音效模式。Hi-Res / SQ(flac) / HQ(320) / 标准(128) 多档音质。

[了解更多 →](/guide/audio)

</div>

<div class="feature-card">

### 🎧 私人 FM

智能推荐个性化电台，根据你的听歌偏好持续优化推荐算法，发现更多好音乐。

[了解更多 →](/guide/personal-fm)

</div>

<div class="feature-card">

### 🎤 听歌识曲

支持麦克风和系统音频捕获，快速识别正在播放的歌曲。不再错过任何一首好听的歌。

[了解更多 →](/guide/music-recognition)

</div>

<div class="feature-card">

### 💬 歌曲详情与评论

查看歌曲完整档案与播放详情，浏览精彩乐评，支持评论楼层跳转，让你的听歌体验更有故事。

[了解更多 →](/guide/song-detail)

</div>

<div class="feature-card">

### 📜 播放历史

记录你的每一段听歌足迹。支持按时间轴浏览、批量选中删除、一键清空，回顾你的音乐旅程。

[了解更多 →](/guide/settings#数据管理)

</div>

<div class="feature-card">

### 🔑 多种登录方式

支持二维码扫码登录与账号密码登录两种方式。v2.2.7 新增账号密码登录，无需手机扫码即可快速登录。

</div>

<div class="feature-card">

### 🖥️ 跨平台支持

完整适配 macOS、Windows 与 Linux 三大桌面操作系统。无论你用什么电脑，EchoMusic 都能完美运行。

</div>

<div class="feature-card">

### ⚙️ 系统深度集成

macOS MPNowPlayingInfoCenter / Windows SMTC / Linux MPRIS 原生系统媒体控制。系统托盘快捷操作、全局快捷键、Windows 任务栏缩略图工具栏、音频设备切换——让 EchoMusic 真正融入你的桌面。

[了解更多 →](/guide/settings)

</div>

<div class="feature-card">

### 🔄 自动更新

内置应用更新检测与下载，支持静默更新。每次打开应用时自动检查新版本，不错过任何新功能。

</div>

<div class="feature-card">

### 🪟 Mini 模式

将播放器缩为一个紧凑的迷你窗口，仅显示封面和基础控制按钮。窗口始终置顶，适合办公或多任务场景。

[了解更多 →](/guide/playback#mini-模式)

</div>

<div class="feature-card">

### 🌐 空间音效

支持导入 IR（脉冲响应）音频文件，实时渲染空间音效。模拟音乐厅、录音棚等不同声学环境，或打造个性化虚拟环绕声。

[了解更多 →](/guide/audio#空间音效)

</div>

<div class="feature-card">

### 🎤 人声 / 伴奏分离

一键切换仅播放人声或仅播放伴奏，已提取的音轨数据会缓存，同一首歌秒切。无论是跟唱练习还是享受纯器乐，自由掌控。

[了解更多 →](/guide/audio#音效模式)

</div>

<div class="feature-card">

### 🌈 全局主题色

跟随封面自动提取主色调、预设主题方案、自定义颜色三种模式。整个应用的色彩随你喜好而变，打造独一无二的视觉风格。

[了解更多 →](/guide/settings#全局主题色)

</div>

<div class="feature-card">

### 📥 外部歌单导入

支持从多个音乐平台导入外部歌单，一键迁移你的收藏。不再受平台壁垒限制，把所有好歌汇聚一处。

</div>

<div class="feature-card">

### 🚀 开机自启

支持开机自动启动并最小化到托盘。打开电脑就有音乐陪伴，无需手动启动应用。

[了解更多 →](/guide/settings#启动与更新)

</div>

<div class="feature-card">

### 🧩 插件系统

类 VS Code 的高自由度插件扩展。在线浏览安装、一键更新、18 款社区插件覆盖美化/歌词/音频/效率。自定义页面、注入样式、创建浮窗、接管音源——让 EchoMusic 无限扩展。

[了解更多 →](/guide/plugins/)

</div>

</div>

## 技术亮点

| 特性 | 说明 |
|------|------|
| 音频引擎 | libmpv，通过 Rust NAPI 进程内嵌入，零延迟直接函数调用 |
| 音质支持 | Hi-Res、SQ(flac)、HQ(320kbps)、标准(128kbps) |
| 音效模式 | 钢琴、人声、伴奏、骨笛、尤克里里、唢呐、DJ、蝰蛇母带、蝰蛇全景声、蝰蛇超清、空间音效(IR) |
| 播放格式 | 标准音频 + LRC/YRC 逐字歌词解析 |
| 平台输出 | macOS (dmg/zip)、Windows (exe/便携版)、Linux (deb/rpm/AppImage/tar.gz) |
