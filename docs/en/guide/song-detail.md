# 💬 Song Details & Comments

EchoMusic doesn't just let you listen — it helps you discover the story behind the song with complete song info, playback details, and insightful comments.

## Song Details

While a song is playing, click the song name or cover to enter the song details page:

### Basic Info

The song details page shows:

| Info | Description |
|------|------|
| Cover | Full-size album cover display |
| Song Name | Currently playing song title |
| Artist | Performer — click to go to artist details |
| Album | Source album — click to view the complete album |
| Duration | Total song length |
| Release Date | Song release date |

### Playback Details

- Real-time playback progress
- Quality indicator (Hi-Res / SQ / HQ / Standard)
- Current play mode

## Song Comments

See what other listeners think about the song and discover more stories behind it.

### Browsing Comments

1. On the song details page, scroll to the **Comments** section
2. Browse popular comments
3. Comments are sorted by popularity — top comments are shown first

### Comment Features

| Feature | Description |
|------|------|
| Popular Comments | Most-liked comments shown first |
| Newest Comments | Reverse chronological order |
| Thread Jump | Click comment floor number to jump to that conversation |
| Interactions | Like comments you find interesting |

### Comment Sorting

- **Popular**: Sorted by likes, showing the most popular comments
- **Newest**: Sorted by post time, showing the latest discussions

## Action Menu

On the song details page, you can:

| Action | Method |
|------|------|
| Play / Pause | Click the play button |
| Add to Queue | Click "Play Next" |
| Favorite Song | Click the ❤️ icon |
| View Artist | Click the artist name |
| View Album | Click the album name |
| Share Song | Copy the song link |

## Shortcuts

| Action | Shortcut |
|------|--------|
| Open song details | Click the song name or cover on the player bar |
| Go back | `Esc` or click the back button |
