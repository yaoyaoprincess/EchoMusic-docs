# 🔍 Search

EchoMusic supports full search across songs, artists, albums, and playlists, helping you quickly find your target content.

## Basic Search

1. Click the **search icon** on the top navigation bar or use the shortcut
2. Enter a keyword (song name, artist name, album name)
3. Results are displayed in real time

## Search Types

The search page has four category tabs at the top to switch search types:

### Song Search

- Enter a song name keyword
- Results show: song name, artist, album, duration
- Click a song to play directly, or add to the play queue

### Artist Search

- Enter an artist name keyword
- Results show matching artists
- Click an artist to enter their detail page with popular songs and albums

### Album Search

- Enter an album name keyword
- Results show album cover, name, artist
- Click an album to view the complete track list

### Playlist Search

- Enter a playlist name keyword
- Results show playlist cover, name, song count
- Click a playlist to view its full content

## Search Tips

| Tip | Description |
|------|------|
| Use full names | Enter the complete song or artist name for higher accuracy |
| Combined search | Supports "artist name + song name" combined search |
| Pinyin search | Partial pinyin fuzzy search supported for some content |

## Search Results Actions

On the search results page, you can:

- **Click a song** to play it directly
- **Right-click a song** to add to the play queue
- **Click the favorite icon** to add to your favorites
- **Click an artist** to view artist details
- **Click an album** to browse album tracks

## Shortcuts

| Action | Shortcut |
|------|--------|
| Open Search | `Ctrl + F` (Windows/Linux) / `Cmd + F` (macOS) |
| Close Search | `Esc`
