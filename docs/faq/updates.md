# 🔄 更新与反馈

### 如何检查更新

EchoMusic 启动时会自动检查更新。也可以手动检查：**设置 → 关于 → 检查更新**。

### 软件更新下载速度很慢 / 无法获取最新版本 / 插件系统一直加载不出最新的

可以在 **设置 → 实验性功能 → GitHub 加速设置** 中配置加速站（详见群公告）。

### 如何反馈问题

- [GitHub Issues](https://github.com/hoowhoami/EchoMusic/issues) — 提交 Bug 或功能建议
- QQ 群：1036693403
- Telegram：[@EchoMusicGroup](https://t.me/+H9vpkAJrDlViZjU1)
