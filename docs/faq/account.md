# 👤 账号与会员

### 是否需要付费

**不需要。** EchoMusic 是完全免费的开源项目。如果你是通过付费渠道获得的，说明被骗了。

### 账号登录后，创建/收藏歌单失败

请检查登录账号是否为**酷狗概念版账号**。如果不是，请注册酷狗概念版账号后重新登录。

### 软件是否支持领取 VIP？

不支持。可以在手机上用酷狗概念版手动签到领取 VIP，如果觉得麻烦可以搜索相关工具。

### 账号安全

- EchoMusic 直连酷狗官方服务器
- 账号数据不经过第三方
- 建议使用强密码保护酷狗账号安全

### 如何注销/切换账号

在 **设置 → 账号** 中选择退出登录，然后重新登录其他账号。
