# 🎵 播放与音质

### 无法播放歌曲

1. 检查网络连接是否正常
2. 确认账号已正常登录
3. 尝试切换音质（部分音质可能需要 VIP）
4. 重启应用

### 播放失败，提示「暂时无法获取可用音源」

1. 请先检查是否为 VIP 会员
2. 如果不是 VIP，可以在手机上用**酷狗概念版**手动签到领取 VIP
3. 如果觉得麻烦，可以上 GitHub 搜索相关自动化签到工具

### 播放卡顿或断断续续

1. 降低音质（设置 → 音质 → 标准）
2. 检查网络连接稳定性和速度
3. 关闭其他占用网络的应用程序

### 无法切换音质

部分歌曲可能只提供特定音质：

- Hi-Res 和 SQ 需要 VIP 账号
- 老歌可能只有标准音质

### 歌词不同步

1. 尝试切换歌词源（如果歌曲有多版本歌词）
2. 在歌词设置中调整歌词延迟
3. 部分歌曲可能没有精确的逐字歌词，会回退到 LRC 歌词
