# 🔧 功能使用

### 听歌识曲无法使用

- **麦克风模式**：确保已授予麦克风权限，环境不要太嘈杂
- **系统音频模式**：确保音频正在系统中播放

### 私人 FM 推荐不准确

FM 的推荐基于你的听歌历史和反馈：

1. 多使用「喜欢」和「不喜欢」按钮
2. 需要一定的听歌积累，推荐才会越来越精准
3. 多听不同类型的歌曲，丰富推荐维度

### 收藏的歌曲不见了

1. 确认是否已登录正确的账号
2. 收藏数据保存在本地，更换设备不会同步
3. 如果清除了应用数据，收藏会丢失

### 可以下载音乐吗

**不支持。** EchoMusic 不会提供音乐文件下载功能，也不会在本地缓存音频文件。
