# 🔍 发现音乐

EchoMusic 提供丰富的音乐发现功能，帮你找到下一首心头好。

## 首页推荐

打开 EchoMusic 后，首页展示个性化的音乐推荐内容：

- **推荐歌单**：根据你的听歌偏好，智能推荐可能会喜欢的歌单
- **新歌推荐**：最新上线的热门歌曲
- **热门歌手**：当前最受欢迎的歌手推荐

## 排行榜

浏览各类音乐排行榜，了解当下最热门的歌曲：

进入「发现」页面，可查看以下榜单：

| 榜单 | 说明 |
|------|------|
| 飙升榜 | 近期播放量飙升的歌曲 |
| 新歌榜 | 最新发布的热门歌曲 |
| 热歌榜 | 综合热度最高的歌曲 |
| 原创榜 | 优秀原创作品排行 |

点击任意榜单进入详情页，浏览完整歌曲列表。

## 歌手

按歌手发现音乐：

1. 进入「发现」→「歌手」页面
2. 按以下维度筛选：
   - **地区**：华语、欧美、日韩等
   - **风格**：流行、摇滚、民谣、电子等
3. 点击任意歌手，进入歌手详情页：
   - 热门歌曲列表
   - 专辑列表
   - 相似歌手推荐

## 专辑

探索完整的音乐专辑：

1. 进入歌手详情页或搜索页面
2. 浏览专辑列表
3. 点击专辑查看：
   - 专辑封面与简介
   - 完整曲目列表
   - 发行信息

## 歌单

海量歌单，总有一款适合你：

- **官方歌单**：编辑精选的歌单推荐
- **分类浏览**：按风格、场景、心情分类
- **歌单详情**：查看完整歌曲列表，一键播放全部

### 歌单操作

| 操作 | 方法 |
|------|------|
| 播放全部 | 点击歌单页的 **播放全部** 按钮 |
| 收藏歌单 | 点击 ❤️ 收藏，保存到「我的歌单」 |
| 分享歌单 | 复制链接分享给朋友 |

## 音乐推荐

EchoMusic 会根据你的听歌历史，持续优化推荐结果：

- 你听的越多，推荐越精准
- 推荐内容会在首页「推荐歌单」中展示
- 私人 FM 提供更沉浸的推荐体验，详见 [私人 FM](/guide/personal-fm)
