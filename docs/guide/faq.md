# ❓ 常见问题

## 安装与启动

### macOS 提示「无法验证开发者」

这是 macOS 的安全机制，不影响正常使用。解决方法：

```bash
# 方法1：终端执行
xattr -cr /Applications/EchoMusic.app

# 方法2：系统设置
# 系统设置 → 隐私与安全性 → 点击「仍要打开」
```

### Windows 提示「Windows 已保护你的电脑」

点击「更多信息」→「仍要运行」即可。

### Linux AppImage 无法运行

确保已赋予可执行权限：

```bash
chmod +x EchoMusic-*.AppImage
./EchoMusic-*.AppImage
```

### 启动后闪退

1. 检查系统版本是否满足最低要求
2. 尝试以管理员/root 权限运行
3. 检查是否有安全软件拦截

## 播放问题

### 无法播放歌曲

1. 检查网络连接是否正常
2. 确认账号已正常登录
3. 尝试切换音质（部分音质可能需要 VIP）
4. 重启应用

### 播放卡顿或断断续续

1. 降低音质（设置 → 音质 → 标准）
2. 检查网络连接稳定性和速度
3. 关闭其他占用网络的应用程序

### 无法切换音质

部分歌曲可能只提供特定音质：

- Hi-Res 和 SQ 需要 VIP 账号
- 老歌可能只有标准音质

### 歌词不同步

1. 尝试切换歌词源（如果歌曲有多版本歌词）
2. 在歌词设置中调整歌词延迟
3. 部分歌曲可能没有精确的逐字歌词，会回退到 LRC 歌词

## 功能问题

### 听歌识曲无法使用

- **麦克风模式**：确保已授予麦克风权限，环境不要太嘈杂
- **系统音频模式**：确保音频正在系统中播放

### 私人 FM 推荐不准确

FM 的推荐基于你的听歌历史和反馈：

1. 多使用「喜欢」和「不喜欢」按钮
2. 需要一定的听歌积累，推荐才会越来越精准
3. 多听不同类型的歌曲，丰富推荐维度

### 收藏的歌曲不见了

1. 确认是否已登录正确的账号
2. 收藏数据保存在本地，更换设备不会同步
3. 如果清除了应用数据，收藏会丢失

## 账号问题

### 是否需要付费

**不需要。** EchoMusic 是完全免费的开源项目。如果你是通过付费渠道获得的，说明被骗了。

### 账号安全

- EchoMusic 直连酷狗官方服务器
- 账号数据不经过第三方
- 建议使用强密码保护酷狗账号安全

### 如何注销/切换账号

在 **设置 → 账号** 中选择退出登录，然后重新登录其他账号。

## 平台限制

### 支持移动端吗

不支持。EchoMusic 专为桌面端打造，不会支持 Android/iOS 等移动平台。

### 支持下载音乐吗

不支持音源下载。EchoMusic 是播放器，不提供音乐文件下载功能。
