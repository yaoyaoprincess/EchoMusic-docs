# 🎤 听歌识曲

EchoMusic 内置听歌识曲功能，支持麦克风收音和系统音频捕获两种模式，帮你快速识别身边正在播放的歌曲。

## 识别模式

### 麦克风识别

适合识别外部环境中的音乐：

1. 点击导航栏的**「听歌识曲」**
2. 确保麦克风权限已开启
3. 将设备靠近音源
4. 点击**「开始识别」**按钮
5. 等待几秒，系统会返回识别结果

适用场景：
- 咖啡厅、商场、餐厅等公共场所
- 聚会中朋友播放的音乐
- 电视、电影中的背景音乐

### 系统音频捕获

适合识别电脑上正在播放的音频：

1. 点击导航栏的**「听歌识曲」**
2. 切换到**「系统音频」**标签
3. 点击**「开始识别」**
4. 系统会捕获当前播放的音频并进行识别

适用场景：
- 识别网页中播放的歌曲
- 识别视频中的背景音乐
- 识别直播中听到的好歌

## 识别结果

识别成功后，你会看到：

- **歌曲名称** — 匹配到的歌曲名
- **歌手** — 演唱者信息
- **封面** — 歌曲专辑封面
- **操作选项**：
  - 🎵 立即播放
  - ➕ 添加到播放队列
  - ❤️ 加入收藏
  - 📋 复制歌曲信息

## 识别历史

所有识别记录会保存在本地，方便随时回顾：

- 在「听歌识曲」页面可查看历史记录
- 支持清除单条或全部历史
- 历史记录仅保存在本地，不会上传到云端

## 使用技巧

| 技巧 | 说明 |
|------|------|
| 减少环境噪音 | 麦克风模式下，尽量靠近音源，减少背景噪音 |
| 确保网络畅通 | 识别需要联网，确保网络连接稳定 |
| 识别片段选择 | 尽量选择歌曲的副歌部分，识别率更高 |
| 音量适中 | 声音太大或太小都可能影响识别准确度 |

::: warning 注意
麦克风识别需要授予麦克风权限。如果在 macOS 上首次使用，系统会弹出权限请求，请选择「允许」。
:::

## 隐私说明

- 识别过程中采集的音频仅用于匹配歌曲指纹
- 音频片段在识别完成后不会保存
- 所有数据传输均通过加密通道
