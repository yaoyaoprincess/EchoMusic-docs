# 💬 歌曲详情与评论

EchoMusic 不仅让你听歌，还能让你了解歌曲背后的故事——完整的歌曲档案信息、播放详情与精彩乐评。

## 歌曲详情

在播放歌曲时，点击歌曲名或封面，进入歌曲详情页面：

### 基本信息

歌曲详情页展示：

| 信息 | 说明 |
|------|------|
| 歌曲封面 | 专辑封面大图展示 |
| 歌曲名 | 当前播放的歌曲名称 |
| 歌手 | 演唱者，点击进入歌手详情页 |
| 专辑 | 所属专辑，点击查看完整专辑 |
| 时长 | 歌曲总时长 |
| 发行时间 | 歌曲发布日期 |

### 播放详情

- 实时播放进度
- 音质标识（Hi-Res / SQ / HQ / 标准）
- 当前播放模式

## 歌曲评论

查看其他听众对这首歌的看法，发现歌曲背后更多的故事。

### 浏览评论

1. 在歌曲详情页滑动到「评论」区域
2. 浏览热门评论
3. 评论按热度排序，热门评论优先展示

### 评论功能

| 功能 | 说明 |
|------|------|
| 热门评论 | 点赞最多的精彩评论优先展示 |
| 最新评论 | 按时间倒序查看最新评论 |
| 楼层跳转 | 点击评论楼层号，快速定位到对应对话 |
| 评论互动 | 对感兴趣的评论进行点赞 |

### 评论排序

- **热门**：按点赞数排序，展示最受欢迎的评论
- **最新**：按发布时间排序，查看最新讨论

## 操作菜单

在歌曲详情页，你可以：

| 操作 | 方法 |
|------|------|
| 播放/暂停 | 点击播放按钮 |
| 添加到队列 | 点击「下一首播放」 |
| 收藏歌曲 | 点击 ❤️ 图标 |
| 查看歌手 | 点击歌手名 |
| 查看专辑 | 点击专辑名 |
| 分享歌曲 | 复制歌曲链接 |

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 打开歌曲详情 | 点击播放栏的歌曲名或封面 |
| 返回上一页 | `Esc` 或点击返回按钮 |
