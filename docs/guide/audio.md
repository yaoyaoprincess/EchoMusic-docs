# 🎚️ 音频设置

EchoMusic 提供专业级的音频调节功能，包括 10 段 EQ 均衡器、多种音效模式和音质选项。

## 音质选择

EchoMusic 支持多档音质，你可以在播放页面选择：

| 音质 | 格式 | 说明 |
|------|------|------|
| **Hi-Res** | 无损 | 最高音质，适合高端音频设备 |
| **DSD** | 臻品 | DSD 高解析度音频，支持的歌曲较少 |
| **SQ** | FLAC | 无损音质，CD 级别 |
| **HQ** | 320kbps | 高品质压缩，兼顾音质与流量 |
| **标准** | 128kbps | 标准音质，适合网络受限环境 |

::: tip 提示
更高音质需要更快的网络速度和更多的流量。在 Wi-Fi 环境下推荐使用 HQ 或 SQ 音质。
:::

## EQ 均衡器

EchoMusic 内置 10 段图形化均衡器，可精确调节各频段。

### 频段说明

| 频段 | 频率范围 | 听感影响 |
|------|----------|----------|
| 第1段 | 31Hz | 极低频，影响低音的深度与震撼感 |
| 第2段 | 62Hz | 低音的下潜感 |
| 第3段 | 125Hz | 低音的力度 |
| 第4段 | 250Hz | 低中频，影响声音的丰满度 |
| 第5段 | 500Hz | 中频，人声的主体部分 |
| 第6段 | 1kHz | 中频，人声的清晰度 |
| 第7段 | 2kHz | 中高频，影响声音的明亮度 |
| 第8段 | 4kHz | 高频，齿音和细节 |
| 第9段 | 8kHz | 高频延伸，空气感 |
| 第10段 | 16kHz | 极高频，泛音与空间感 |

### 使用方法

1. 进入 **设置 → 音频 → EQ**
2. 拖动各频段滑块调节
3. 实时试听效果

::: tip 调音建议
- 增强低音（31Hz-125Hz）：适合电子、嘻哈
- 增强中频（500Hz-1kHz）：突出人声
- 增强高频（4kHz-16kHz）：增加细节和空气感
- 建议小幅调整（±3dB 以内），避免失真
:::

## 音效模式

EchoMusic 内置 9 种音效模式，一键切换不同听感：

| 音效 | 适用场景 |
|------|----------|
| **人声** | 仅播放人声，削弱伴奏 |
| **伴奏** | 仅播放伴奏，削弱人声 |
| **钢琴** | 突出钢琴音色，适合纯音乐 |
| **人声伴奏** | 削弱人声，突出伴奏 |
| **骨笛** | 增强高频，适合民乐 |
| **尤克里里** | 明亮轻快的音色 |
| **唢呐** | 增强穿透力 |
| **DJ** | 增强低频，适合电子舞曲 |
| **蝰蛇母带** | 母带级音频处理 |
| **蝰蛇全景声** | 空间环绕音效 |
| **蝰蛇超清** | 高清音频还原 |

### 空间音效

EchoMusic 支持通过 IR（脉冲响应）音频文件实现空间音效渲染：

1. 在 **设置 → 音效 → 空间音效** 中导入 IR 音频文件
2. 支持重命名和移除已导入的 IR 文件
3. 在播放页面的音效弹出层中快速切换不同空间音效

适用场景：
- 模拟不同声学环境（音乐厅、录音棚等）
- 虚拟环绕声体验
- 耳机空间音频增强

::: tip 提示
你可以在网上搜索或录制自己的 IR 文件，EchoMusic 会实时渲染到播放音频中。
:::

在播放页面点击音效按钮即可切换。

## 音量均衡

基于 LUFS（Loudness Units relative to Full Scale）响度标准化：

- 自动调整不同歌曲的响度至相近水平
- 避免切歌时音量突变
- 在 **设置 → 音频** 中开启

## 音频设备

管理音频输出设备：

1. 点击播放栏的音频设备图标
2. 选择要使用的输出设备（扬声器、耳机等）
3. 支持**独占模式**：绕过系统混音，直接输出到设备，减少延迟

::: warning 注意
独占模式下，其他应用无法同时播放音频。如需同时使用多个音频应用，请关闭独占模式。
:::
