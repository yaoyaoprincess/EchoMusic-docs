# 📝 歌词系统

EchoMusic 拥有强大的歌词显示系统，支持多种歌词格式与展示模式。

## 歌词格式支持

| 格式 | 说明 |
|------|------|
| **LRC** | 标准歌词格式，逐行时间轴同步，兼容性最广 |
| **YRC** | 逐字歌词格式，精确到每个字的同步显示 |

> 当歌曲同时存在 LRC 和 YRC 歌词时，默认优先使用 YRC 逐字歌词，提供更精准的跟唱体验。

## 歌词显示模式

EchoMusic 提供多种歌词显示模式，满足不同场景需求。

### 播放栏歌词

在底部播放栏中显示当前歌词行，紧凑简洁。适合日常使用，不占额外屏幕空间。

### 全屏歌词

点击播放栏的歌词区域或播放页面的「全屏歌词」按钮：

- 全屏沉浸式歌词体验
- 歌词居中显示，跟随播放进度平滑滚动
- 支持自定义背景和字体大小
- 点击空白区域或按 `Esc` 退出

### 桌面歌词

让歌词悬浮在桌面任意位置，不影响其他操作：

1. 在设置中开启「桌面歌词」
2. 歌词悬浮窗始终置顶显示
3. 可拖拽调整位置
4. 支持调整字体大小和颜色

::: tip 提示
桌面歌词特别适合在工作、浏览网页时使用，随时查看当前歌词。
:::

## 歌词设置

进入 **设置 → 歌词**，可调整以下选项：

| 设置项 | 说明 |
|------|------|
| 歌词字体大小 | 调整歌词显示字号 |
| 歌词对齐方式 | 左对齐 / 居中对齐 |
| 歌词延迟 | 微调歌词与音乐的时间同步 |
| 桌面歌词颜色 | 自定义桌面歌词文字颜色 |
| 桌面歌词背景 | 开启/关闭桌面歌词背景框 |
| 全屏歌词背景 | 选择全屏歌词的背景效果 |

## 歌词选择

当一首歌曲存在多个版本的歌词时（如不同语言翻译、不同来源等），EchoMusic 支持手动选择：

1. 在歌词页面点击**歌词选择**按钮
2. 从可用歌词列表中选取你想要的版本
3. 选择后即刻切换显示

## 歌词过滤

支持使用正则表达式过滤歌词内容：

1. 进入 **设置 → 歌词 → 歌词过滤**
2. 输入正则表达式规则（如 `/纯音乐|伴奏/` 过滤纯音乐标记行）
3. 匹配的内容将在歌词显示中隐藏

适用场景：
- 过滤歌词中的广告或推广信息
- 隐藏纯音乐标记行
- 去除不需要的特殊符号或注释

## 翻译歌词

部分歌曲支持翻译歌词显示：

- 当歌曲有翻译歌词时，会在原文下方显示译文
- 可在歌词设置中开启/关闭翻译显示
- 支持多语言翻译（需歌曲本身提供）

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 全屏歌词 | 点击播放栏歌词区域 |
| 退出全屏歌词 | `Esc` |
| 开启/关闭桌面歌词 | 在设置中切换 |
