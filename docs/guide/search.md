# 🔍 搜索

EchoMusic 支持歌曲、歌手、专辑、歌单全方位搜索，帮你快速定位目标内容。

## 基本搜索

1. 点击顶部导航栏的**搜索图标**或使用快捷键
2. 输入关键词（歌名、歌手名、专辑名）
3. 系统实时展示搜索结果

## 搜索类型

搜索页面顶部有四个分类标签，可以切换搜索类型：

### 歌曲搜索

- 输入歌名关键词
- 结果列表展示：歌曲名、歌手名、专辑名、时长
- 点击歌曲直接播放，或添加到播放队列

### 歌手搜索

- 输入歌手名关键词
- 结果展示匹配的歌手列表
- 点击歌手进入详情页，查看热门歌曲和专辑

### 专辑搜索

- 输入专辑名关键词
- 结果展示专辑封面、名称、歌手
- 点击专辑查看完整曲目

### 歌单搜索

- 输入歌单名称关键词
- 结果展示歌单封面、名称、歌曲数量
- 点击歌单查看完整内容

## 搜索技巧

| 技巧 | 说明 |
|------|------|
| 使用完整名称 | 输入完整歌名或歌手名可提高精确度 |
| 组合搜索 | 支持「歌手名 + 歌名」组合搜索 |
| 拼音搜索 | 部分内容支持拼音模糊搜索 |

## 搜索结果操作

在搜索结果页面，你可以：

- **点击歌曲** 直接播放
- **右键歌曲** 添加到播放队列
- **点击收藏** 将歌曲加入收藏列表
- **点击歌手** 查看歌手详情
- **点击专辑** 浏览专辑曲目

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 打开搜索 | `Ctrl + F`（Windows/Linux）/ `Cmd + F`（macOS） |
| 关闭搜索 | `Esc`
