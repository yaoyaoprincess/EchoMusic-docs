# 🎧 私人 FM

私人 FM 是 EchoMusic 的智能电台功能，根据你的听歌偏好自动推荐音乐，帮你发现更多好歌。

## 什么是私人 FM

私人 FM 像一个专属的音乐电台：

- 无需手动选歌，系统自动连续播放
- 基于你的听歌历史和偏好智能推荐
- 越听越懂你，推荐越来越精准
- 适合不想费心选歌、只想放松听音乐的场合

## 如何使用

1. 在首页或侧边栏点击**「私人 FM」**
2. FM 播放器自动开始播放推荐歌曲
3. 享受连续不断的音乐流

## 交互操作

在 FM 模式下，你可以对当前播放的歌曲做出反馈：

| 操作 | 图标 | 说明 |
|------|------|------|
| 喜欢 | ❤️ | 标记喜欢，系统会推荐更多类似歌曲 |
| 不喜欢 | 🚫 | 跳过当前歌曲，系统会减少此类推荐 |
| 跳过 | ⏭ | 切换到下一首推荐歌曲 |
| 暂停 | ⏸ | 暂停 FM 播放 |

你的每一条反馈都会帮助 FM 更好地了解你的音乐品味。

::: tip 提示
对不喜欢的歌曲点击「不喜欢」，FM 会立即跳过并减少类似风格的推荐。这是训练 FM 品味的最快方式。
:::

## FM 与普通播放的区别

| 特性 | 私人 FM | 普通播放 |
|------|---------|---------|
| 选歌方式 | AI 自动推荐 | 手动选择 |
| 播放列表 | 无限流式推荐 | 固定播放队列 |
| 反馈机制 | 喜欢/不喜欢 | 无 |
| 适用场景 | 随意听歌、发现新歌 | 听特定歌曲 |
| 个性化 | 持续学习优化 | 无 |

## 小技巧

- **刚使用 FM 时**：多使用「喜欢」和「不喜欢」来训练推荐算法
- **长时间使用 FM**：推荐会越来越精准，碰到喜欢的歌会越来越多
- **切换模式**：FM 中听到喜欢的歌曲，可以点击歌曲名进入详情页或添加到收藏
