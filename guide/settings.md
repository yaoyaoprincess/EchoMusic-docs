# ⚙️ 个性化设置

EchoMusic 提供丰富的个性化选项，让你把播放器调成自己喜欢的样子。

## 外观主题

EchoMusic 支持多种外观模式，适应不同使用场景：

| 主题 | 说明 |
|------|------|
| 🌞 浅色模式 | 明亮清爽，适合白天使用 |
| 🌙 深色模式 | 护眼舒适，适合夜间或暗光环境 |
| 🔄 自动切换 | 跟随系统设置自动切换深浅模式 |

在 **设置 → 外观** 中切换主题。

### 全局主题色

EchoMusic 支持三种主题色模式：

| 模式 | 说明 |
|------|------|
| 跟随封面 | 自动从当前歌曲封面提取主色调，打造视觉一致的色彩体验 |
| 预设主题 | 从内置配色方案中选择 |
| 自定义颜色 | 手动选择你喜欢的颜色 |

在 **设置 → 外观 → 主题色** 中切换模式。

### 字体设置

支持自定义应用字体：

- 在 **设置 → 外观 → 字体** 中选择系统已安装字体
- 调整字体大小
- 预览效果实时生效

## 快捷键

### 全局快捷键

全局快捷键在应用后台运行时仍然生效：

| 操作 | 默认快捷键 |
|------|-----------|
| 播放/暂停 | 媒体键 ▶⏸ |
| 上一首 | 媒体键 ⏮ |
| 下一首 | 媒体键 ⏭ |
| 音量+ | 媒体键 🔊 |
| 音量- | 媒体键 🔉 |

### 应用内快捷键

| 操作 | 默认快捷键 |
|------|-----------|
| 打开搜索 | `Ctrl + F` / `Cmd + F` |
| 播放/暂停 | `空格` |
| 关闭弹窗 | `Esc` |

在 **设置 → 快捷键** 中可自定义快捷键。

## 系统托盘

EchoMusic 支持最小化到系统托盘，在后台持续播放：

- 关闭窗口时，EchoMusic 最小化到托盘而非退出
- 右键托盘图标，弹出快捷菜单：

| 操作 | 说明 |
|------|------|
| 播放/暂停 | 快速控制播放 |
| 上一首 / 下一首 | 快速切歌 |
| 显示主窗口 | 打开应用主界面 |
| 退出 | 完全退出应用 |

- 支持 **Mini 模式**：将播放器缩为紧凑迷你窗口，点击标题栏按钮切换

在 **设置 → 系统** 中调整托盘行为。

## 启动与更新

| 设置项 | 说明 |
|------|------|
| 开机自启 | 系统启动时自动运行 EchoMusic |
| 启动后最小化 | 启动后自动最小化到托盘 |
| 启动时自动播放 | 启动后自动开始播放上次的歌曲（v2.2.7 新增，默认关闭） |
| 自动更新 | 后台自动检查并下载新版本 |
| 静默更新 | 下载完成后下次启动自动安装 |

## 数据管理

### 播放历史

v2.2.7 新增了独立的播放历史页面，你可以：

- 浏览完整的听歌记录
- 批量选中并删除不需要的记录
- 一键清空全部历史

在侧边栏或导航中可以找到播放历史入口。

### 本地存储

EchoMusic 使用本地 SQLite 数据库存储：

- 播放历史
- 收藏列表（歌曲、歌单）
- 播放队列状态
- 个人偏好设置

::: info 提示
所有个人数据仅保存在本地，不会上传到云端，保护你的隐私安全。
:::

### 数据清除

在 **设置 → 数据** 中可清除：

- 播放历史
- 搜索历史
- 缓存数据

## 播放设置

| 设置项 | 说明 |
|------|------|
| 淡入淡出时长 | 切歌时的过渡效果时长（0-5秒） |
| 默认音量 | 启动时的默认音量 |
| 启动时自动播放 | 启动后自动继续上次播放 |

## 开发者工具

EchoMusic v2.2.7 新增了实验性 **DevTools 开关**，面向开发者：

- 在 **设置 → 开发者** 中开启
- 开启后可访问浏览器开发者工具，调试页面和插件

::: warning 注意
DevTools 为实验性功能，普通用户无需开启。
:::
