# 🎵 播放控制

EchoMusic 提供专业级的播放控制能力，让你以最舒适的方式享受音乐。

## 基础操作

### 播放/暂停

- 点击底部播放栏的 **播放/暂停** 按钮
- 按键盘 **空格键**（需要应用处于焦点状态）
- 使用系统媒体键（macOS 触控栏、键盘媒体键）

### 切歌

- **上一首**：点击播放栏的 ⏮ 按钮
- **下一首**：点击播放栏的 ⏭ 按钮
- 使用系统媒体快捷键

### 音量调节

- 拖动播放栏的**音量滑块**调整音量
- 点击音量图标快速**静音/取消静音**
- 使用系统音量键同步调节

### 进度控制

- 在播放栏的**进度条**上点击或拖动，跳转到任意位置
- 当前播放时间和总时长会实时显示

## 播放队列

播放队列管理你当前和即将播放的歌曲列表。

### 队列操作

| 操作 | 方法 |
|------|------|
| 添加到队列 | 在歌曲列表中右键 → **添加到播放队列** |
| 排队候播 | 在歌曲列表中右键 → **排队候播**（插播优先） |
| 移除歌曲 | 在队列中右键 → **移除** |
| 清空队列 | 点击队列底部的 **清空** 按钮 |
| 调整顺序 | 在队列中拖拽歌曲调整播放顺序 |

### 排队候播

在歌曲列表中右键点击歌曲，选择「**排队候播**」，该歌曲会被追加到下一首播放队列的末尾，优先于队列中的其他歌曲播放。适合临时插播想听的歌，不影响当前队列顺序。

多次排队候播的歌曲按操作顺序排列，全部播完后恢复原队列顺序。

::: tip 提示
播放队列会保存在本地，下次启动应用时会自动恢复。
:::

## 播放模式

EchoMusic 支持多种播放模式，点击播放栏的模式按钮切换：

| 模式 | 图标 | 说明 |
|------|------|------|
| 顺序播放 | ▶️ | 按队列顺序播放，播完后停止 |
| 列表循环 | 🔁 | 按顺序播放队列，播完后从头循环 |
| 单曲循环 | 🔂 | 重复播放当前歌曲 |
| 随机播放 | 🔀 | 随机打乱队列顺序播放 |

## 倍速播放

支持从 0.5x 到 2.0x 的倍速播放：

1. 点击播放栏的**倍速**按钮
2. 选择想要的播放速度
3. 1.0x 为正常速度

适用场景：
- **0.5x - 0.75x**：学习歌曲细节、跟唱练习
- **1.25x - 1.5x**：快速浏览歌曲
- **2.0x**：快速过歌

## 淡入淡出切歌

EchoMusic 支持平滑的淡入淡出切换效果：

- 切歌时，当前歌曲音量渐低，下一首歌曲音量渐高
- 营造自然的过渡效果，减少突兀感
- 可在 [音频设置](/guide/audio) 中调整淡入淡出时长

## Mini 模式

Mini 模式将播放器缩为一个紧凑的迷你窗口，适合在后台听歌时节约屏幕空间：

- 点击标题栏的 **Mini 模式** 按钮切换
- Mini 窗口仅显示封面、歌名歌手和基础播放控制
- 窗口始终置顶，可拖拽调整位置
- 点击 Mini 窗口可恢复完整界面

适用场景：
- 办公时不想被播放器占太多屏幕
- 配合桌面歌词，最小化视觉干扰
- 多任务场景下保持音乐控制随手可达

## 任务栏缩略图工具栏

Windows 平台下，将鼠标悬停在任务栏 EchoMusic 图标上，会显示缩略图工具栏，可快速控制：

- ⏮ **上一首**
- ▶ / ⏸ **播放/暂停**
- ⏭ **下一首**

无需切换窗口即可控制音乐播放。

## 启动时自动播放

v2.2.7 新增了「启动时自动播放」设置（默认关闭）。开启后，每次启动 EchoMusic 会自动开始播放上次的歌曲。可在 [个性化设置](/guide/settings#启动与更新) 中配置。

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 播放/暂停 | 空格（应用焦点时）/ 系统媒体键 |
| 下一首 | 系统媒体键 |
| 上一首 | 系统媒体键 |
| 音量+ / 音量- | 系统音量键 |

更多快捷键设置请参考 [个性化设置 - 快捷键](/guide/settings#快捷键)。
